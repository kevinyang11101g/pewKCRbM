# 前言

欢迎来到我们的企业信息管理系统项目！该项目基于SSM（Spring、Spring MVC、MyBatis）框架，以Java为主要开发语言，结合前端技术Vue、JS和CSS3，旨在为企业提供一个高效、易用、安全的信息管理解决方案。以下是项目的详细介绍。

# 内容介绍

本项目是一款基于B/S架构的企业信息管理系统，主要功能包括企业基本信息管理、员工信息管理、部门管理、权限管理等。系统采用模块化设计，具有良好的可扩展性和易维护性。通过使用前沿的技术栈，我们致力于为用户提供一个友好、便捷的操作界面，确保系统的高效运行。

# 技术介绍

// 引入MyBatis Mapper接口
import com.example.mapper.EnterpriseMapper;

@Service
public class EnterpriseService {

    @Autowired
    private EnterpriseMapper enterpriseMapper;

    public List<Enterprise> getAllEnterprises() {
        return enterpriseMapper.selectAllEnterprises();
    }
}
